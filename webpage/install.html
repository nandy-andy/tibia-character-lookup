How to install it?

The extension is in development phase. Therefore the only way to install it is to do it manually.

Once the development work is finished the extension and add-on will be officially available on Chrome Web Store and official Mozilla Add-ons page.

Google Chrome

Download the newest version of extension from here. Unpack it somewhere on your disk and then enable it in Google Chrome following these instructions.

You will notice a small icon while browsing the Internet:

Clicking it will take you to this website but it also means the extension is enabled and in your context menu you will find a new item "Lookup Tibia character...".

Mozilla Firefox

Just click this link to download the newest version of add-on. A small pop-up will appear which will ask you to Allow my page to install the add-on. After clicking "Allow" a new window will appear when you click "Install Now".

After a while the installation will complete and you will be able to use the add-on!
